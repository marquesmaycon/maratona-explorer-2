# RocketLinks

![HTML](https://img.shields.io/badge/HTML5-E34F26?style=for-the-badge&logo=html5&logoColor=white)
![CSS](https://img.shields.io/badge/CSS3-1572B6?style=for-the-badge&logo=css3&logoColor=white)
![Rocketseat](https://img.shields.io/badge/Maratona_Explorer-2-8257E5?style=for-the-badge)

Interface no estilo Linktree criada durante a Maratona Explorer 2 da Rocketseat. O projeto centraliza links pessoais e redes sociais em uma pagina simples, responsiva e visualmente direta.

## Demo

[Acessar projeto online](https://marquesmaycon.github.io/maratona-explorer-2/)

## Recursos

- Avatar e identificacao do perfil.
- Links para LinkedIn, GitHub, WhatsApp e e-mail.
- Tema escuro.
- Layout centralizado e responsivo.
- Icones externos para destacar cada link.

## Stack

| Tecnologia | Uso |
| --- | --- |
| HTML | Estrutura da pagina |
| CSS | Estilizacao |
| Phosphor Icons | Icones |
| Google Fonts | Tipografia |

## Como executar

Abra o arquivo `index.html` no navegador.

## Estrutura

```text
.
|-- index.html
`-- assets/
    |-- images/
    `-- styles/
```
